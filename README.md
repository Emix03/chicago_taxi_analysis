# Chicago Taxi Trip Analysis

## Project Overview

This project analyzes taxi trip patterns in Chicago to understand market structure and evaluate whether weather conditions affect travel duration to O'Hare International Airport.

## Objectives

* Identify dominant taxi companies in the market
* Analyze demand distribution across neighborhoods
* Evaluate the effect of rainy Saturdays on trip duration

## Dataset

The dataset includes:

* Company trip counts
* Neighborhood average drop-offs
* Airport trips with weather conditions

## Methods

* Exploratory Data Analysis (EDA)
* Data visualization
* Hypothesis testing (Welch’s t-test)

## Key Findings

Bad weather significantly increases trip duration to the airport on Saturdays.

## Tools

Python, Pandas, Matplotlib, SciPy, Jupyter Notebook
